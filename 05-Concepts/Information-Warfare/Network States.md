# Network States

## Definition

Political-theoretical framework articulated by **[[Balaji Srinivasan]]** in his 2022 book **"The Network State: How to Start a New Country"**. Argues that **online communities with shared values can become new political entities**:

1. Online community forms with shared moral / political framework
2. Community builds shared institutions (governance, economic systems)
3. Community crowdfunds physical territory acquisition
4. Network state eventually gains diplomatic recognition

## Conceptual ancestors

Network States synthesizes earlier Tech Right intellectual currents:
- [[Patri Friedman]]'s seasteading (build new states in international waters)
- Charter city movement (Friedman's later focus)
- Crypto-anarchism (governance via cryptographic / blockchain mechanisms)
- Curtis Yarvin's "exit" framework (leave failing institutions rather than reform)
