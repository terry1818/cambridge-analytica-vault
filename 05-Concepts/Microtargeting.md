# Microtargeting

Marketing and political-targeting methodology delivering customized message content to individuals based on detailed individual-level data profiles.

## Standard data inputs

- Voter file (party registration, voting history, donation history)
- Commercial data (purchases, magazine subscriptions, vehicle registration)
- Online behavioral data (browsing, social media engagement)
- Demographic data (Census, ACS)
- Psychographic overlay where available ([[Psychographics]])

## Pre-CA history

- Karl Rove's 2004 Bush re-election was the first widely-discussed US presidential application
- Obama 2008 and 2012 campaigns substantially scaled the digital-targeting layer
- CA's 2016 Trump work added the psychographic overlay but did not invent the underlying microtargeting

## Significance for vault

The vault treats microtargeting as the methodological substrate that [[Cambridge Analytica]] / [[Internet Research Agency]] / [[Team Jorge]] all built on. The political question is not whether microtargeting exists - it is who controls the data and what content gets delivered to whom.
