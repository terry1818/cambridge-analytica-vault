# State Actor MOC

The State Actor cluster is the foreign-state and state-aligned-commercial information-operations infrastructure that intersects with the CA-era and present-day vault network. The cluster is operationally distinct from the principally-US-domestic Religious-Right, Patron, and Tech-Right wings, but it interacts substantially with them.
