## Method

- Each file holds a verbatim-as-captured clean-text snapshot, with YAML frontmatter recording source_url, source_tier, captured date, and a wayback locator.
- Two independent preservation layers: the local copy (survives deletion/censorship of the original) and the Wayback Machine URL (survives loss of this vault).

## Honest caveats

- Capture completeness: snapshots are clean-text extractions -- article body and tables are preserved; images, scripts, and interactive elements are not. The Wikipedia applications list is captured in full; very long aggregator pages are captured to a high character cap.
- The Wayback locator is a pointer, not a guarantee -- it resolves to an existing snapshot if one exists, or offers Save Page Now if not. Triggering fresh Wayback captures for each URL is a recommended follow-up.
- Copyright: these are personal research archive copies. Fine for a private vault; do not republish. Full books are not archived here -- only citations and excerpts in their source notes.
- Live registries (Companies House, Ballotpedia, COSAction) update continuously -- each archive is a dated point-in-time snapshot, not a live mirror.
