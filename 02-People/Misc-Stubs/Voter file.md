# Voter file

Public record of registered voters maintained by each US state. Contains name, address, registration date, party affiliation (in registered-party states), and voting-history data (turnout, not vote choice).

## Vault relevance

Voter files are the foundational data layer for US political microtargeting. Both parties' data operations ([[Data Trust]] for Republicans, NGP VAN for Democrats) build on top of voter file data.

[[Cambridge Analytica]]'s methodological innovation was overlaying psychographic profiles on otherwise-conventional voter file data. The substantive data layer was conventional; the personality-trait scoring was the (controversial) novel addition.
