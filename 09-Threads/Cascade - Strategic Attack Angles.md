# Cascade — Strategic Attack Angles

> Companion to [[Deep Dive - Three Wing Coalition Anatomy]], [[Counter-COS Strategy for the Left]], and [[Counter-Architecture]]. This file translates the three-wing structure into the specific operational angles by which Cascade can be deployed.

## What Cascade is

Cascade is not a directory + not a newsletter + not a petition platform. Cascade is a **daily-action routing engine** that:

1. Identifies the **single highest-leverage action a specific user can take today** given their location, capacity, skills, and what's live in the threat surface
2. **Routes coordinated pressure** at the right target at the right moment
3. **Aggregates individual actions** into collective documented pressure
4. **Exploits seams** between the three wings rather than fighting all three uniformly
5. **Compounds attention** by routing amplification into post-action propagation

Cascade's structural advantage: the right wing's coordination machinery ([[Council for National Policy]], [[Project 2025]], [[Conservative Partnership Institute]], [[Heritage Foundation]]) routes its operators to the right target at the right moment. **Cascade is the operational equivalent for the counter-coalition.** The asymmetry documented in [[Counter-Architecture]] (10-20x weaker by funding / personnel / state coverage) means Cascade must out-execute by routing precision rather than mass.

## Gaps in current Cascade build

The cascade-prototype.html v1 has **5 tabs** (today / threats / graph / amplify / me). To operationalize the angles above, the following gaps need filling in v2:

### Data gaps
- **Live state-legislation tracker** (LegiScan API) for state-by-state daily-action routing
- **Federal-legislation tracker** (Congress.gov + GovTrack API) for Senate/House daily-action routing
- **FCC + IRS + FTC docket tracker** (regulations.gov API) for comment-period actions
- **Litigation case tracker** (CourtListener API) for plaintiff-recruitment + amicus signature actions
- **Sheriff + school-board + judicial elections calendar** (Ballotpedia API)
- **Flock Safety deployment tracker** (EFF Atlas of Surveillance + crowd-sourced)
- **CSPOA-trained sheriff tracker** (PRA + adjacent research)

### Feature gaps
- **Coalition partner integration** ([[Democracy Alliance]] + Indivisible + MoveOn + Working Families + adjacent orgs) so Cascade routes users to existing coalition action rather than reinventing
- **Action-completion tracking** with aggregate-pressure visualization (collective-pressure feedback loop)
- **Donor/subscription routing** — for angles that depend on small-dollar litigation funding or local-journalism subscriptions
- **Wing-specific filter** — let users pick which wings they want to focus on (patron / theocratic / tech / coordination keystones / paramilitary)
- **Personal-network amplification** — let users tag personal-network members for specific share-targets (different message for evangelical aunt vs Tech Right friend vs progressive cousin)
- **Skills routing** — lawyers route to amicus-brief signatures; teachers route to school-board engagement; healthcare workers route to IVF/medical-access pressure; tech workers route to internal-organizing
- **Geographic precision** — congressional district + state legislative district + county commission + city council resolution (currently only state-level)

### Coordination gaps
- **CNP-member exposure database** integration ([[CNP State-by-State Affiliate Map]] data)
- **Heritage / P2025 partner-org tracker** with action surfaces
- **Federalist Society judicial nominee tracker** (specifically lower-court + administrative-law-judge positions)
- **Project 2025 implementation tracker** — what has happened vs what is planned
