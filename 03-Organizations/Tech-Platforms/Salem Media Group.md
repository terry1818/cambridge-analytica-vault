# Salem Media Group

## Summary

**Conservative + Christian radio / media network** founded 1986. Operates ~99 radio stations + multiple websites + book publishing. **Major distribution platform for the theocratic wing** - hosts Christian + conservative + Christian-nationalist programming including the **Charlie Kirk Show** (Salem distributed). **CNP-affiliated**: co-founders Atsinger + Epperson were CNP members; Salem operates as **the radio-distribution arm of CNP coordination**.

## Identity

- **Founded:** 1986
