# The Network State (concept and book)

Book published 2022 by [[Balaji Srinivasan]]. Articulates the framework that crypto-enabled online communities can crystallize into political entities that eventually negotiate sovereignty with existing nation-states.

## Core thesis

- Existing nation-states are legacy structures imposed by 19th-20th-century geography
- Internet-native communities of shared values can form coherent populations
- Once populations are large enough, they can negotiate territorial / sovereignty arrangements
- The model is bottom-up secession via market-rate land acquisition rather than top-down regime change

## Vault relevance

The Network State is one of the most-developed contemporary articulations of the [[Tech right]] "exit" framework - the alternative to "capture the existing state" that [[Curtis Yarvin]]'s neoreaction articulates. Both reject conventional democratic competition within existing US political structures.

[[Prospera]] (Honduras charter city) is the most-developed real-world implementation of Network-State-adjacent principles.
