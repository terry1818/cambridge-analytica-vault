## Evidence tiers â€” read this before quoting anything

Every claim is labeled by evidence strength. **The tier is the disclaimer.**

- **Tier 1 â€” Documented.** Court filings, FBI files, FEC records, leaked corporate documents, sworn testimony with corroboration, peer-reviewed research. Citable directly.
- **Tier 2 â€” Inferred from solid evidence.** Reasonable inferences from Tier-1 facts, mainstream investigative journalism with named sources, multiple independent reportings. Cite with attribution.
- **Tier 3 â€” Contested / single-source / under-evidenced.** Single-witness claims, allegations without corroboration, plausible patterns without smoking guns. **Cite as allegation, not fact.**

If a note doesn't carry a tier, the wider claim usually inherits the tier of its weakest [[Source: ...]] citation. See [Evidence Tiers](00-Home/Evidence%20Tiers.md) for the full framework and [Methodology - Verifying a CA claim](00-Home/Methodology%20-%20Verifying%20a%20CA%20claim.md) for the verification protocol.
